# Mnemonic 忆存线 — v3.5.0

> 三省引擎(CMG) 状态记忆层。读取 Guard 拦截日志，自动识别高频错误模式。
> **v3.5.0 命中上下文保留。**

## 核心能力

- **模式识别**：同会话 ≥2 次 → 自动推送规则草稿至 Canon
- **数据源降级**：Guard拦截日志 → Canon错误记录 → 等待状态
- **误报率自适应**：置信度 ±0.1/0.2 浮动

## v3.5.0 新增

### 命中上下文保留
- 每次 Guard 拦截命中记录触发场景（用户意图 + Agent操作）
- 解决 unknown 规则无法还原的问题（33次命中上下文全丢）
- unknown 规则再次触发时自动基于上下文生成规则草稿
- 最近 100 条保留，超限归档，每 rule_id 最多 5 条
- 隐私保护：只记操作摘要，不记完整对话

### 已有能力
- 同会话 2 次推草稿（v3.4.0）
- 数据源降级链（v3.3.0）
- 独立持久化 + 误报率调节（v3.2.0）

## 安装

```bash
npx skills add mnemonic --yes --global
```
